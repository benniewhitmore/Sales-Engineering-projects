# Sales Engineering Projects

Python projects built to demonstrate API integration, data automation, and real-world sales use cases.

Built by Bennie Whitmore — former enterprise AE transitioning into Sales Engineering.

---

## Projects

| # | Project | What It Does | Tech |
|---|---------|-------------|------|
| 1 | [Inventory Alert System](./01-inventory-alert) | Monitors stock levels, sends WhatsApp alerts via API when items drop below threshold | Python, Twilio, CSV |
| 2 | [Weather Briefing Tool](./02-weather-briefing) | Pulls live weather for each rep's territory before a call block | Python, OpenWeatherMap API, CSV |
| 3 | [Lead Prioritization Tool](./03-lead-prioritization) | Categorizes pipeline leads as HIGH/MEDIUM/LOW based on deal value | Python, CSV |
| 4 | [Account Enrichment Tool](./04-account-enrichment) | Combines CRM data with live weather API to produce enriched account briefings | Python, OpenWeatherMap API, CSV |
